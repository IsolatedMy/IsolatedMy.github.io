#### 使用std::atomic报错

参考[论坛](https://stackoverflow.com/questions/27314485/use-of-deleted-function-error-with-stdatomic-int)。如下使用`std::atomic<size_t>`类型时

```c++
std::atomic<size_t> SomeClass::m_job_cnt = 0;
```

发生错误

```
use of deleted function 'std::__atomic_base<_IntTp>::__atomic_base...
```

根据论坛上的解释。我的代码尝试在右值构造一个暂时的`std::atomic_int`，接着使用`std::atomic_int`的拷贝构造器(已被删除)去初始化`m_job_cnt`，所以没办法成功。
